# App Cardapio

Aplicativo mobile simples feito com React Native, Expo Router e TypeScript.

## Funcionalidade

- Cadastro de produtos com nome, preco e descricao.
- Listagem dos produtos cadastrados na tela.

## Requisitos

- Node.js 18 ou superior
- npm

## Como executar

```bash
npm install
npm start
```

Depois, escolha onde abrir o app pelo terminal do Expo.

## Estrutura principal

```text
app/
  _layout.tsx
  index.tsx
src/
  screens/
    ProductFormScreen.tsx
  types/
    Product.ts
```
